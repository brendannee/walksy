Credits

Walksy.com was created for the Google I/O Last Call Maps/Geo Round 2 Challenge by Brendan Nee. The source code is available on github.

If you like this app, you might like some of the other transportation web apps Brendan has created:

  • Bikesy.com - customized bike route directions for the San Francisco Bay Area based on data from OpenStreetMap
  • Modepick.com - calculate the time, distance and cost for a trip including walking, biking, carshare, transit, taxi, Amtrak and Greyhound for any two points in the US.
